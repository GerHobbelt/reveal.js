# What's Bitcoin?
## Project * Started as a 20% project at Google * First Rev: March 4th, 2011 * Main Devs: [Mike Hearn](https://plus.google.com/u/0/114798402540078632611), [Miron Cuperman](http://hyper.to/blog/), [Matt Corallo](https://github.com/TheBlueMatt) * Apache 2.0 Licensed * Still actively developed
↠ BitcoinJ Website ↞
## Goals * Full Java implementation of the Bitcoin protocol * Lightweight client for the Android ecosystem * Simple library for desktop applications
## Main Features
  • Manage a local Bitcoin wallet (no server!)
  • Connect and listen to other Bitcoin nodes directly
  • Send/recieve Bitcoins w/o storing the full blockchain (simplified payment verification)
