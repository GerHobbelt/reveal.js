# Know the pitfalls ![pitfalls](images/banana_peel.png)
## Implied globals Forgetting var ```javascript function foo() { bar = 1; } foo(); ```
## Boolean type conversion To Truthy or to Falsy. That is the *only* question! ```javascript var el = document.getElementById('does_not_exist'); if(el == false) { alert("shouldn't we see this message?!"); } ```
## Trailing comma works on my machine! ```javascript var foo = { bar: "bar", baz: "baz", }; ```
## Return undefined señor developers wear mustaches { ```javascript var foo = function() { return { x : "looks like C# now!" }; } ```
## Associative arrays they don't exist ```javascript var x = []; x['foo'] = "bar"; ```
## try .. catch .. finally who cares about the reason? ```javascript var foo = function() { try { doCrazyStuff; } catch (e) { return false; } return true; }; ```
## Eval ... and the job is done ```javascript function poorMansJsonParser(text) { return eval("(" + text + ")"); } var text = ' { "hello" : "world" } '; var json = poorMansJsonParser(text); ```
## Eval is evil! Never ever! ```javascript var text = ' function() { alert("hacked!"); })( '; ```
