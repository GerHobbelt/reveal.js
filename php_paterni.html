Factory patern

Jedan od najčešće korišćenih paterna

Objekti se kreiraju bez otkrivanja logike kreiranja klijentu

Factory klasa

            
class Mailbox_Display_Factory
{
    /**
    * Returns new instance of inbox display printer
    *
    * @param string $type
    * @param array $config
    */
    public static function forgeDisplayPrinter($type, array $config = array())
    {
        $class_name = '\\Application\\Message\\Mailbox_Display_'.ucfirst($type);

        return new $class_name($config);
    }
}

Strategy klase

            
class Mailbox_Display_Abstract
{
    public function __construct(array $config = array())
    {
        $this->config = $config;
    }

    public function render()
    {
        $this->prepare_data();
        $this->view = View::forge('messages/templates/messages/wrapper/'.$this->view_name, $this->view_data, false);

        return $this->view->render();
    }

    protected function prepareData()
    {
        return false;
    }
}

Themes

Reveal.js comes with a few themes built in:
Sky - Beige - Simple - Serif - Night - Default

* Theme demos are loaded after the presentation which leads to flicker. In production you should load your theme in the <head> using a <link>.

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the background.

"blackout"

"soothe"

Custom Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


                        Reveal.addEventListener( 'customevent', function() {
                        console.log( '"customevent" has fired' );
                        } );
